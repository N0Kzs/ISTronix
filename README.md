# ISTronix

## Project Overview

ISTronix is a web-based retail platform dedicated to the sale of electronic products and components. This project is the Final Project repository for the course **CIS 226 - Programming for E-Business Applications**.

## Features

- **Dynamic Product Catalog**: Browse and search through a comprehensive collection of electronic products and components
- **User-Friendly Interface**: Streamlined design optimized for an intuitive online shopping experience
- **E-Commerce Functionality**: Complete retail platform with product management and purchasing capabilities

## Technology Stack

- **PHP** (93.9%) - Server-side logic and backend functionality
- **CSS** (3.4%) - Styling and user interface design
- **Hack** (2.7%) - Additional scripting and utilities

## About This Course

This repository contains the coursework for **CIS 226 - Programming for E-Business Applications**, focusing on building functional e-commerce solutions using modern web technologies.
